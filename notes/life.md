This site is generated by a single Python script that ingests itself. The script reads its own source code, pulls named regions into this document, and renders a page that contains the generator's own annotated code. The generator is a subject of the site it generates.

That's the quine property.

## what an entry is

{@region: entry_model}

Shape and medium aren't labels attached after the fact — they're the first things declared, and they determine how an entry looks before you read it. A seed is italic and receded. A complete essay has weight. An event is upright and dim. The typography carries the taxonomy.

## collecting from git

Every commit to this repository becomes an entry in the stream. The collector reads the raw git plumbing format so it gets the full commit body, not just the subject line.

{@region: collect_git}

The topics heuristic is minimal — conventional commit prefixes like `feat(topic):` and explicit footer lines like `Topics: a, b`. Enough to start building facets. Over time the facets reflect what the work actually touches.

## collecting notes

Markdown files in the `notes/` directory become prose entries. The front matter carries shape, medium, and topic. The body is processed for `{@region}` directives before it reaches the markdown renderer — which is how this document you're reading pulls live code from `quine.py` without modifying either file.

{@region: collect_notes}

The `annotates` key in front matter tells the directive processor which source file to look in when a directive doesn't specify a path explicitly. This document annotates `quine.py`, so `{@region: collect_git}` pulls from there.

## rendering

Bodies are markdown. Before the markdown library sees them, fenced code blocks are syntax-highlighted by Pygments, which gives richer language-specific colour than the markdown library's built-in highlighter.

{@region: render_entry_body}

The page template lives inside the script itself — no external files. The generator is self-contained. Download the script and you have everything.
